# ARM - Monocycle microarchitecture

supported functions
- Data processing
  - ADD
  - SUB
  - AND
  - ORR
  - EOR
  - LSL
  - LSR
  - ASR
  - ROR
  - MOV
  - TST
  - TEQ
  - CMP
  - CMN
- Memory
  - STR
  - STRB
  - LDR
  - LDRB
- Brench
  - all condictions
- **Frame supported**
  - Immediate
  - Register
  - Register-shifted Register
